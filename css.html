Different types of selectors

  1. Element selector (sometimes called a tag or type selector)-All HTML elements of the specified type.
  2. ID selector-The element on the page with the specified ID. On a given HTML page, each id value should be unique.
  3. Class selector-The element(s) on the page with the specified class. Multiple instances of the same class can appear on a page.
  4. Attribute selector-The element(s) on the page with the specified attribute.
  5. Pseudo-class selector-The specified element(s), but only when in the specified state. (For example, when a cursor hovers over a link.)

There are many more selectors to discover. To learn more, see the MDN Selectors guide.
